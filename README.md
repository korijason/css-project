# css-project

## Project Setup Instructions

Welcome to the project! This README file will guide you through setting up and customizing the styles for our HTML project. Follow the steps below to ensure everything is correctly configured.

### Step 1: Create `styles.css`

1. Create a new file in your project directory named `styles.css`.

### Step 2: Link `styles.css` to Your HTML

1. Open your HTML file ( `index.html`).
2. Link the `styles.css` file by adding the following line within the `<head>` section of your HTML document:

    ```html
    <link rel="stylesheet" href="styles.css">
    ```

### Step 3: Modify Navbar Display

1. Open the `styles.css` file.
2. Add the CSS to change the display of the navbar to inline:

  .bar{
    display: inline;
    }

### Step 4: Change the Font to Roboto

1. Import the Roboto font by adding the following line to the top of your `styles.css` file:
<link rel="preconnect" href="https://fonts.googleapis.com">
<link rel="preconnect" href="https://fonts.gstatic.com" crossorigin>
<link href="https://fonts.googleapis.com/css2?family=Roboto:ital,wght@0,100;0,300;0,400;0,500;0,700;0,900;1,100;1,300;1,400;1,500;1,700;1,900&display=swap" rel="stylesheet">
   


### Step 5: Add Class Attributes to HTML

1. Go through your HTML file and add class attributes to the elements you want to style.
   
   Example:
    
    <nav class="bar">
